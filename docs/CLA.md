- **Cédant / Auteur** : Kevin Uhlmann

## 2. Objet
Le présent CLA définit les conditions selon lesquelles le Cédant cède ses droits sur le code et contenus de l’application « Église Chrétienne à Proximité ».

## 3. Droits cédés
- Le Cédant reste propriétaire du code et contenus.

- Toute reproduction, distribution, modification ou usage commercial nécessite autorisation écrite de Kevin Uhlmann.

## 4. Garanties
- Le Cédant garantit qu’il est titulaire des droits et que le code est original.  
- Le Cédant ne cède aucun droit moral.  

## 5. Entrée en vigueur
